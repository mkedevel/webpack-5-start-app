### Каталог для разработки

```tree
src
├── assets
│   ├── fonts (шрифты файлы)
│   └── images (изображения)
│── styles
│   ├── colors.scss (цвета)
│   ├── fonts.scss (шрифты)
│   ├── includes.scss (подключение всех зависимостей)
│   └── style.scss (базовые стили)
│── utils
│   └── utils.js (вспомогательные функции)
│── index.html (entry template)
└── index.js (entry point)
```
